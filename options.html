Default sorting:

Allows: name, unread>0, started, (tag). Repeat to reverse. e.g. name fantasy started started

Update frequency:

Form: "x <time>". time can be any of day(s), hour(s), minute(s), second(s). e.g. 3 days Update frequency global minimum: Your browser and some websites may not like you sending a lot of requests at the same time. Form: same as above.

Rules:

Form: JSON string. {"match":___ ,"matchExcludes":___ ,"selector":___ ,"tags":[___]} Match and selector required. Match is a regexp that shows which pages to save. e.g. "match":".+:\/\/example.com\/.+" Selector is a css selector that finds the element containing the links you want. e.g. {"match":".+:\/\/example.com\/.+","matchExcludes":".+:\/\/example.com\/.+\/frogs.+","selector":".listing","tags":["example","not frogs"]}
