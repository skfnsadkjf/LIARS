Default sorting:

Allows: name, unread>0, started, (tag). Use ! for reverse order. e.g. name fantasy !started

Update frequency:

Form: "x <time>". time can be any of day(s), hour(s), minute(s), second(s). e.g. 3 days
Update frequency global minimum: Your browser and some websites may not like you sending a lot of requests at the same time. Form: same as above.

Rules:

Form: JSON string. { match:___ , matchExcludes:___ , selector:___ , tags:[___] } Match and selector required. Match is a url that may contain * as a wildcard. e.g. *example.com* Selector is a css selector that finds the element containing the links you want. e.g. { match:"*example.com*" , matchExcludes:"*example.com/frogs/*", selector:".listing" , tags:["example","not frogs"] }
